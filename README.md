# arnavs-macropad

A simple macropad built of 5 switches and 2 LEDs.
The switches are in a row design to allow quicker pressing of the buttons.

BOM:

- 4x Cherry MX Switches
- 4x Blank DSA Keycaps
- 2x SK6812 MINI Leds
- 1x XIAO RP2040
- 1x Case (Top and Botton)
- 4x Bolts
